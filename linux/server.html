1. External drive
    • sudo vim /etc/fstab
    • UUID=767E-687A /mnt/external exfat umask=0022,gid=1000,uid=1000 0 1
    • Find UUID with: lsblk -f
  2. Samba
    • Arch Wiki
    • sudo pacman -S samba
    • sudo vim /etc/samba/smb.conf
      • # fix log file (see wiki) log file = /var/log/samba/%m.log [Media] path = /mnt/external force user = iggames browsable = yes guest ok = yes writable = yes read only = no
    • sudo systemctl restart smb.service
    • sudo systemctl enable smb.service
