1. External drive
    • sudo vim /etc/fstab
    • UUID=767E-687A /mnt/external exfat defaults 0 1
    • Find UUID with: lsblk -f
  2. Samba
    • Arch Wiki
    • sudo pacman -S samba
    • sudo vim /etc/samba/smb.conf
      • [MEDIA] path = /mnt/external browsable = yes guest ok = yes writable = yes read only = no
    • sudo systemctl restart smb.service
    • sudo ufw allow samba
